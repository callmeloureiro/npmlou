## Getting Started

```bash
# Clone it:
$ git clone https://github.com/callmeloureiro/npmlou

# Then, go to the project's folder:
$ cd npmlou

# Install your dependencies and Init a new project by running:
$ npm i && npm start
```

## Running the tests

```bash
npm test
```
